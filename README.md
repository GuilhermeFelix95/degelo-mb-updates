# Degelo MB Updates

Repositório público para releases automáticas do Degelo MB.

- Código principal: https://github.com/GuilhermeFelix95/degelo-mb.git
- Releases/atualizações: https://github.com/GuilhermeFelix95/degelo-mb-updates.git

Este repositório é usado pelo app para buscar e baixar atualizações.

## Como funciona
1. O app verifica novas versões neste repositório.
2. Releases são publicadas automaticamente via GitHub Actions.

## Contato
Dúvidas/sugestões: abra uma issue ou envie email para o mantenedor.
